Telegram-бот с локальной LLM

🎯 Цель
Разработать Telegram-бота, который работает как простой AI-чат, используя локальную языковую модель (LLM).

🧠 Описание задачи
Необходимо создать Telegram-бота, который:

- принимает текстовые сообщения от пользователя
- передает их в локальную LLM
- возвращает сгенерированный ответ обратно в Telegram

❗ Важно:

- Каждое сообщение должно обрабатываться независимо, без хранения истории диалога.

⚙️ Технические требования

1. LLM (обязательно локально)
   Использовать систему:

- Ollama

Использовать модели:

- qwen3.5:0.8b
- deepseek-r1:1.5b


2. Язык программирования

- Python


3. Telegram API
   Использовать Telegram Bot API через:

- polling
  НЕ webhook!


4. Архитектура
   Минимальная архитектура:
   Telegram → Bot → LLM → Bot → Telegram
   Без базы данных и без хранения состояния.

🤖 Особое требование (ключевое)

- Код бота должен быть сгенерирован с помощью LLM (например, ChatGPT, локальная модель, Copilot и т.д.).
- Реализовать обработку ошибок
- Реализовать логирование
- обработку start команды
- настройку системного промпта
- выбор модели через команду
- логирование запросов
- необходимо создать .gitignore файл и добавить туда секреты, файлы логов и т.д.
- секреты должны подтягиваться из виртуального окружения, должен быть .env файл в котором их можно указать (при запуске
  проекта данные должны подтягиваться оттуда), файл должен быть в .gitignore
- использовать aiogram
- код должен обрабатываться асинхронно
- необходимо покрыть код тестами

Разрешается:

- генерировать код частями
- дорабатывать и исправлять

Но запрещено:

- писать весь код вручную с нуля

📦 Функциональные требования
Бот должен:
✅ отвечать на любые текстовые сообщения
✅ корректно обрабатывать ошибки (например, если LLM недоступна)
✅ работать стабильно при нескольких запросах подряд

🚫 Ограничения
❌ Без хранения истории диалога
❌ Без использования облачных LLM (OpenAI, Anthropic и т.д.)
❌ Без базы данных
❌ Не должно быть необработанных ошибок

💡 Подсказка
Минимальный pipeline:

- Получить сообщение из Telegram

- Вернуть ответ пользователю
